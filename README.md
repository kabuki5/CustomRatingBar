# CustomRatingBar


Usage example:

<com.kabuki5.logicalratingbar.CustomRatingBar
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        app:iconSize="58dp"
        app:maxItems="5"
        app:clickable="true"
        app:initRating="3"
        app:imageFull="@drawable/ic_heart_full"
        app:imageEmpty="@drawable/ic_heart_empty"
        />
